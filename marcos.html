Preguntas y Respuestas

¿Cuál es la estructura básica de las tablas?
Se compone de <table>, <tr>, <td> y <th>.

¿Cuál es la estructura básica de los formularios?
Se usa <form> con <input>, <label>, <textarea> y <button>.

¿Cuál es la estructura básica de los marcos?
Actualmente se usa <iframe>.

¿Qué significa DHTML?
Significa Dynamic HTML.

¿Qué es un sitio Web?
Es un conjunto de páginas web alojadas en un servidor.

¿Cómo se crea un marco?
Con la etiqueta <iframe>.

¿Para qué se utilizan los temas?
Para definir el diseño visual del sitio.

¿Cómo se crea un sitio web?
Creando archivos HTML, aplicando CSS y subiéndolos a un servidor.
